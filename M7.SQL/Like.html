LIKE: câu lệnh dùng để tìm kiếm

Cú pháp : SELECT column1, column2, ...FROM table_name WHERE columnN LIKE pattern; Ký hiệu MySQL % đại diện cho 0 ,1 hoặc nhiều ký tự _ đại diện cho 1 ký tự ví dụ: WHERE CustomerName LIKE 'a%' Tìm mọi giá trị của tên khách hàng bắt đầu là chũ 'a' -SELECT * FROM customers WHERE first_name LIKE 'a%' -'a%' : chữ a đứng đầu -'%a' : chữ a ở cuối -'a__%' Bắt đầu bằng chữ 'a' và có ít nhất 3 ký tự -'a%o': Bắt đầu bằng chữ 'a' và kết thúc bằng chữ 'o'
