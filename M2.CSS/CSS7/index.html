Border

Border styte dotted (kiểu các dấu chấm )

Border styte dashed (kiểu các gạch đứt quãng )

Border styte solid (kiểu các đường liển mạch )

Border styte double (kiểu 2 đường liền mạch chạy song song)

Border styte groove(kiểu 3d)

Border styte ridge(kiểu 3d mỏng)

Border styte inset(kiểu đổ bóng trong)

Border styte outset(kiểu đổ bóng ngoài)

Border styte none(kiểu không có gì)

Border styte hidden(kiểu có border nhưng ẩn đi)

Border styte mix(kiểu có border dạng kết hợp nhiều border)
