Box-Sizing

BoxSizing có giá trị mặc định là content-box

Content-Box => thuộc tính width trong CSS là độ rộng của content

Content-Box => thuộc tính height trong CSS là độ cao của content

border-box => thuộc tính width trong css là tổng độ rộng của box model

border-box => thuộc tính height trong css là tổng độ cao của box model
