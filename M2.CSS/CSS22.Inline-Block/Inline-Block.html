CSS display: inline-block

KHi nào sử dụng inline block

Trường hợp 1: Khi muốn đặt thuộc tính width height có tác dụng trên các thẻ có kiểu hiển thị mặc định là inline như a hay span thì ta chuyển thuộc tính display: inline-block;

Các thẻ a và span có kiểu hiển thị là display: inline-block nằm trên cùng 1 hàng giống thẻ inline

Và đặt được width và height giống thẻ block

Trường hợp 2: là mình dùng các thẻ block hiển thị trên cùng 1 hàng giống thẻ inline
