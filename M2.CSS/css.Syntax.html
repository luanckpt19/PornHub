CSS Syntax ( cú pháp ) and selector ( bộ chọn)

CSS rule-set consists of a selector and a declaration block:

style="color: red" > một quy tấc cứ pháp css bao gồm 1 selector và một khối khai báo code của nó

The selector points to the HTML element you want to style.

style="color: red" > selector sẽ chỉ ra thẻ html mà bạn muốn trang trí

The declaration block contains one or more declarations separated by semicolons.

style="color: red"> Một đoạn khai báo code chứa một hay nhiều đoạn khai báo mã css phân tấch bởi dấu

p>Each declaration includes a CSS property name and a value, separated by a colon.

Mỗi đoạn khai báo bao gồm thuộc tính css và giá trị phân tách bởi dấu :

A CSS declaration always ends with a semicolon, and declaration blocks are surrounded by curly braces.

Đoạn khai báo css luôn kết thúc với dấu ; và khối khai báo được bao quanh bởi {}

Ví dụ về code css : 

        p {
          color: red;
          text-align: center;
        }
